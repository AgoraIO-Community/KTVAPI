# KTVAPI

### 概述

为降低开发者的集成难度，声网为 K 歌房场景提供了 K 歌场景化 API, 后文简称 KTVAPI。KTVAPI 封装了声网音视频 SDK 的 API，并提供了 K 歌业务常见的功能，例如，获取版权歌单、下载歌曲、对领唱和伴唱进行 NTP 时间同步等。你只需要调用简单的 KTVAPI 组合即可轻松实现 K 歌场景。
